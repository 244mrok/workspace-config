# workspace-config

開発環境のワークスペース設定とClaude Codeガイド。

## 概要

複数の独立したプロジェクトを含むワークスペースの設定リポジトリです。

| プロジェクト | 技術スタック | 概要 |
|---|---|---|
| bitflyer-autotrade-ai | Python, FastAPI, CCXT | bitFlyer向け暗号資産自動売買ボット (BTC/JPY) |
| retailnext-ai | Python, FastAPI, OpenSearch, OpenAI | AI搭載リテールデモアプリ |
| mcp-server-quickstart | TypeScript, MCP SDK | Model Context Protocolサーバーテンプレート |
| fitness_nextjs_clean | Next.js, React | フィットネスプログラムサイト |

## 含まれるファイル

- **CLAUDE.md** — [Claude Code](https://claude.ai/code)向けのワークスペースガイド（ビルド手順、アーキテクチャ概要）
